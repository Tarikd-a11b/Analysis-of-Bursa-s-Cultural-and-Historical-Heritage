# Analysis-of-Bursa-s-Cultural-and-Historical-Heritage
The tourist attractions of Bursa were analyzed using Argis Pro

Bursa Digital Heritage & GIS Project 🏛️🛰️
This project focuses on the digital preservation and spatial analysis of Bursa’s UNESCO World Heritage sites using ArcGIS Pro and Web-GIS technologies. Developed as a part of the Management Information Systems (MIS) curriculum at Izmir Bakircay University.

📌 Project Overview
Bursa, the first capital of the Ottoman Empire, hosts a vast array of historical landmarks. This project transforms qualitative historical data into a quantitative spatial database to analyze the distribution and preservation status of these heritage sites.

🛠️ Tech Stack & Skills
GIS Software: ArcGIS Pro.

Web-GIS: ArcGIS Online & StoryMaps.

Data Management: Attribute Table Management, Schema Design.

Analysis: Spatial Analysis and Functional Distribution.

Data Science Tools: Potential for Python integration (ArcPy) for further automation.

🚀 Key Features & Implementation
Manual Digitization: Precise digitizing of 15+ landmarks, thematic routes, and social complexes (külliye) using "Point, Line, and Polygon" layers from high-resolution satellite imagery.

Database Design: Implementation of a structured Attribute Table containing information such as protection status, construction period, and functional categories.

Interactive Storytelling: Integration of spatial data with multimedia through ArcGIS StoryMaps for a comprehensive digital experience.

Modern vs. Historic Analysis: Inclusion of modern landmarks like Timsah Arena to analyze the urban transition between historical heritage and modern architecture.
